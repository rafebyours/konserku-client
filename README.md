**Konserku** adalah aplikasi mobile untuk pemesanan tiket konser yang dikembangkan menggunakan **Vue.js**, **Tailwind CSS**, dan **Firebase** di sisi frontend, serta **Flask (Python)** dan **MongoDB** untuk backend dan database. Aplikasi ini memungkinkan pengguna untuk melakukan registrasi, login, melihat daftar konser, serta memesan tiket secara online.

---

## 🚀 Fitur Utama

- 🔐 Registrasi dan Login pengguna
- 🎫 Pemesanan tiket konser
- 📋 Daftar event konser dengan detail
- 🔄 Update data secara real-time dengan Firebase
- 🗃️ Manajemen data pengguna dan tiket
- 🧾 Integrasi API antara frontend dan backend
- 🧠 Desain responsif dan mobile-friendly

---

## 🛠️ Teknologi yang Digunakan

### Frontend
- Vue.js
- Tailwind CSS
- Firebase (Realtime Database & Auth)

### Backend
- Flask (Python)
- MongoDB
- REST API

### Tools
- Visual Studio Code
- Git & GitHub
- Postman (untuk pengujian API)
- Figma (UI Design)
